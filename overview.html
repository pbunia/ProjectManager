This program is a tool that allows it's user to manage efficiently with tasks in busy environment of any kind especially designed for office daily work. This tool allows to streamline the workload to user with clear deadline indicating with proper color an approaching finish date. Green if the finish date is above 5 days, yellow color if finish date is within 3 to 5 days, in red when there 2 and less days remaining to finish date. Program uses 4 methods GET POST PUT DELETE to communicate with server. This is an early stage of the program. Further developments will continue.
